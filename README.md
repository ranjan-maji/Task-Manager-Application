# Task Manager & Employee Directory

A full-stack application for managing tasks and employee information built with React.js, Node.js, Express.js, and MongoDB.

## Features

### Task Manager
- Create, read, update, and delete tasks
- Mark tasks as completed
- Search tasks by title
- Pagination support
- Task status tracking (pending, ongoing, completed)

### Employee Directory
- Add, edit, and delete employees
- Search employees by email
- Filter by department
- Pagination support
- Employee details with optional image

## Tech Stack

- **Frontend**: React.js, Tailwind CSS, React Router, Axios
- **Backend**: Node.js, Express.js
- **Database**: MongoDB with Mongoose
- **Additional**: React Icons, date-fns

## Prerequisites

- Node.js (v14 or higher)
- MongoDB (local or Atlas)
- npm or yarn
